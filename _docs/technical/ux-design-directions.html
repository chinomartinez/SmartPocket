Decisiones de Diseño

Tabs Gasto/Ingreso

Cambio de tipo en 1 clic. Estado visual claro (rojo=gasto, verde=ingreso). Sin radio buttons.

Monto Protagonista

Font 22px monospace, símbolo $ pre-fijo. Es el dato más importante → mayor tamaño visual.

Botón 🔢 Mini Calc

Visible pero no invasivo. On-demand → no estorba el flujo básico. Color accent sutil.

Campos Opcionales

Descripción y Tags al final. Label indica "(opcional)". No bloquean el flujo rápido.

Smart Defaults

Fecha = Hoy, Cuenta = última usada, Tipo = Gasto (más frecuente). Mínimas decisiones.

Diseño de Mini Calculadora

Layout 4×5 Estándar

Idéntico al patrón conocido (Gestor de Gastos, calculadora del sistema). Cero curva de aprendizaje.

Display Superior

Muestra operación en curso (200+100+300) y resultado. 2 líneas = historial + número actual.

Botón = Destacado

Color Primary Blue (como amarillo en Gestor de Gastos). Alta visibilidad, confirmar la operación.

Botón "Usar Resultado"

Inserta $600,00 al campo monto automáticamente. Color éxito (verde) indica confirmación.

Panel Integrado

Emerge debajo del campo monto dentro del mismo modal. Sin modal anidado. Scroll natural.

Caso de Uso

Merienda: 200 (galletas) + 100 (dulce de leche) + 300 (jugo) = 600. Click "Usar resultado" → $600 al campo monto.

Decisiones de Navegación Mobile

Bottom Navigation (5 items)

Inicio · Transacciones · Cuentas · Dashboard · Más (⋯). "Más" despliega: Próximos Pagos, Categorías, Configuración.

Header Simplificado

Sin sidebar en mobile. Header con logo + notificaciones. El sidebar desktop desaparece en viewport pequeño.

FAB Posicionado sobre Bottom Nav

FAB a bottom:72px para no solapar la barra de navegación. Accesible con pulgar derecho.

Formulario: Pantalla Completa

En mobile el formulario de transacción es pantalla completa (no modal flotante). Más espacio para tocar.

Sin Disparidad Funcional

Mismas features que desktop. Solo cambia el layout. Mini calculadora, dropdowns, tags: todos accesibles.

Breakpoints Propuestos

• Mobile (<768px): Bottom nav + pantalla completa forms

• Tablet (768-1024px): Sidebar colapsable (hamburger)

• Desktop (>1024px): Sidebar fijo + modal forms
